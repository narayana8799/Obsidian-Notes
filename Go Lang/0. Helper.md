Place Holders
--
| Place Holder | Data Type                    |
| ------------ | ---------------------------- |
| %d           | Integer                      |
| %o           | Octal (Integer)              |
| %x           | Hex (Integer)                |
| %f           | Float's with no exponent     |
| %g           | Compact Floats               |
| %e           | Float's with exonent         |
| %s           | Strings                      |
| %t           | Boolean                      |
| %q           | quoted strings               |
| %v           | Default format for any value |
| %p           | Pointers                     |
